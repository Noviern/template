# Template d'addon

Point de départ minimal pour les nouveaux addons.

- Fenêtre principale
- Fenêtre de paramètres (reconnue par le Gestionnaire d'extensions https://github.com/Noviern/manager)
- Support des locales / traductions

Il suffit de cloner/télécharger → renommer → coder.
