# Аддон-шаблон

Минимальная отправная точка для создания новых аддонов.

- Главное окно
- Окно настроек (распознаётся Менеджером дополнений https://github.com/Noviern/manager)
- Поддержка локалей / переводов

Просто клонировать/скачать → переименовать → писать код.
